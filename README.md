🚀 Smart Personal Finance Dashboard (AI powered dashboard)

A high-performance, AI-driven financial tracking application built with a focus on **AI delegation**. Over 90% of this project was generated using **Cursor Composer**, leveraging the latest in LLM-assisted development.

## 🌟 Key Features
- **Glassmorphism UI:** A sleek, professional dark-mode interface with frosted-glass effects.
- **Dynamic Charting:** Real-time visualization of spending habits and net worth trends using Chart.js.
- **AI Insights Engine:** Automated analysis of transaction data to provide spending alerts and financial recommendations.
- **Responsive Design:** Fully optimized for desktop and mobile viewing.

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS (Glassmorphism design language)
- **Icons:** Lucide React
- **Charts:** Chart.js + React-Chartjs-2
- **Deployment:** Vercel
- **AI Tooling:** Cursor Composer + @Web (for design benchmarking)

## 🚀 Getting Started

### Prerequisites
